# Java Core, Concurrency & Systems Practice

Данный репозиторий содержит учебные Java-проекты, выполненные в рамках интенсивного 4-месячного курса с акцентом на современные инженерные практики.
Основное внимание уделяется **Java Core**, структурам данных, многопоточности, анализу производительности и низкоуровневому системному программированию.

---

## Изученные темы

*   **Java Core & ООП**
*   **Collections Framework** (пользовательские реализации)
*   **Алгоритмы и анализ сложности**
*   **Многопоточность и параллелизм**
*   **Java Reflection API**
*   **Бенчмаркинг производительности**
*   **ExecutorService и пулы потоков**
*   **Сокетное программирование и веб-серверы**
*   **Git workflow** (ветки, PR)

---

## Структура проектов

### **HW01 – Пользовательский List & Collections Framework**
*   Динамический массив с реализацией интерфейса `List`
*   Unit-тесты, сравнивающие поведение с `ArrayList`
*   Обобщённая реализация (Generics)
*   Сравнение производительности со стандартными коллекциями JDK
*   Освоение основ Git и рабочего процесса

**Ключевые темы:** Java Collections, Generics, JUnit, Тестирование производительности, Git

---

### **HW02 – Алгоритмы, Lombok & Reflection**
*   **Реализации чисел Фибоначчи:**
    *   Рекурсивная
    *   С мемоизацией
    *   Итеративная
*   Анализ временной и пространственной сложности
*   Сравнение производительности и потребления памяти
*   Сравнение производительности операций с массивами
*   Пользовательский раннер тестов на основе Java Reflection API

**Ключевые темы:** Алгоритмы, Big-O анализ, Lombok, Reflection, Фреймворки для тестирования

---

### **HW03 – LinkedList, Stack & Queue**
*   Пользовательская реализация `LinkedList`
*   Реализация `Stack`, `Queue` и `Deque` на основе собственных структур
*   Сравнение производительности пользовательских реализаций списков

**Ключевые темы:** Структуры данных, абстракция, анализ производительности

---

### **HW04 – HashMap & Concurrent Maps**
*   Пользовательская реализация `HashMap`, реализующая интерфейс `Map<K, V>`
*   Стратегии разрешения коллизий
*   Unit-тесты и сравнение производительности с `HashMap` из JDK
*   Потокобезопасная реализация карты (Map)
*   `ConcurrentSkipListMap` (задача повышенной сложности)

**Ключевые темы:** Хеширование, многопоточность, потокобезопасность, производительность

---

### **HW05 – Многопоточность и синхронизация**
*   Параллельная обработка массивов
*   Сравнение платформенных и виртуальных потоков
*   Симулятор банковской системы с конкурентными переводами
*   **Механизмы синхронизации:**
    *   `synchronized`
    *   `ReentrantLock`
    *   атомарные операции
*   Потокобезопасные декораторы для пользовательского списка
*   Создание гарантированных сценариев взаимной блокировки (deadlock)

**Ключевые темы:** Управление многопоточностью, синхронизация, виртуальные потоки, состояния гонки

---

### **HW06 – Пользовательский ExecutorService & Веб-сервер**
*   Пользовательская реализация `ExecutorService`
*   Поддержка платформенных и виртуальных потоков
*   Очередь задач и корректное завершение работы
*   Многопоточный веб-сервер на основе сокетов
*   Обслуживание статического и динамического контента
*   Сравнение производительности с нагрузочным тестированием

**Ключевые темы:** Пул потоков, виртуальные потоки, сетевое программирование, HTTP, проектирование систем
